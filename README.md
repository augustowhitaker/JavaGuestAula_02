# Supermercado em Java

## Descrição

Sistema simples de supermercado desenvolvido em Java com Swing.
Permite adicionar produtos a um carrinho, calcular o valor total da compra
e aplicar automaticamente um desconto promocional.

## Funcionalidades

- Adicionar produtos ao carrinho por meio de botões
- Cálculo automático do valor total
- Aplicação de desconto quando o total atinge o valor mínimo

## Tecnologias

- Java
- Swing

## Como Executar

1. Clone o repositório
2. Abra o projeto na sua IDE
3. Execute a classe principal `App.java`
